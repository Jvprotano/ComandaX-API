## About the Project

ComandaX is a backend API for a restaurant management system, built with .NET and using a Clean Architecture approach. It features a GraphQL API for managing tables, customer tabs, products, and orders. The project is structured into four main layers: `Domain`, `Application`, `Infrastructure`, and `WebAPI`.

For a more detailed overview of the project, please refer to the `AI_PROJECT_SUMMARY.md` file.
